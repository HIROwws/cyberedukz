# CyberEdu KZ Design Direction

## Product Feel

CyberEdu KZ should feel like a focused cybersecurity learning workspace: dark, practical, structured, and motivating.

The design is closer to TryHackMe and Hack The Box Academy than to a marketing landing page.

## Visual Principles

- Dark interface by default.
- High contrast text for readability.
- Teal/green accent for progress, success, and primary actions.
- Blue accent for learning tracks and navigation.
- Compact cards for courses and challenges.
- Clear status labels: Beginner, Easy, Medium, Completed, In Progress.
- No heavy decoration that distracts from learning tasks.

## Core Screens

- Home dashboard
- Registration
- Login
- Course catalog
- Course page
- Challenge page
- Profile
- Leaderboard

## MVP UI Priority

1. Navigation and layout
2. Auth pages
3. Course cards
4. Challenge workspace
5. Progress indicators
6. Leaderboard

## Tone

Russian-first interface with clear cybersecurity terms. Kazakhstan/CIS adaptation should appear in content examples, course naming, and learning context rather than heavy decorative national styling.
